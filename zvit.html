Тема: Каскадні таблиці стилів (CSS)

Опис виконання

У проєкті використано селектори тегів, класів, ідентифікаторів, таблиці, списки, форми. CSS підключено окремим файлом style.css.

HTML-код (фрагмент)

<h1 id="main-title">Ретро Ігри Онлайн</h1>

CSS-код (фрагмент)

#main-title {
  color: #c0392b;
  text-align: center;
}

Скріншот сайту

Контрольні питання

  1. Каскадні таблиці стилів використовуються для оформлення HTML-документів.
  2. Методи: inline, embedded, external.
  3. Каскадування — черговість стилів, спадкування — передача властивостей дочірнім елементам.
  4. Селектор тега: p {}, класу: .class {}
  5. ID використовується для унікального стилю: #id
  6. Оформлення: кольори, шрифт, фон, таблиці, списки

Висновок

У цій лабораторній я навчилася використовувати CSS для стилізації HTML-елементів, працювати з різними селекторами, і створювати адаптивні блоки на сторінці.
